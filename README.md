# Neon Snake Arena

A modern arcade-style Snake game with power-ups, enemy AI, and boss battles.

![Game Screenshot](screen%20shots/game_playing.png)

## Features

- **4 Unique Power-Ups:**
  - 👻 Ghost Mode - Pass through enemies and walls
  - 🧲 Magnet - Pulls food from distance
  - 💊 POWERPILL - Destroy enemies on contact
  - ⏱️ Slow Down - Slows enemy movement

- **Boss Battle Mode:** Press 'B' to challenge the giant boss snake

- **Lives System:** 3 lives with spawn protection

- **Combo Multiplier:** Chain food eats for score bonuses

- **Kill Streaks:** Track consecutive kills

- **Screen Effects:** Shake, flash, particles, and floating text

## Controls

| Key | Action |
|-----|--------|
| C | Start Game |
| Arrow Keys | Move |
| Z/X | Speed Up/Down |
| P | Pause |
| R | Restart |
| B | Boss Battle Mode |
| V | Toggle Sound Set |

## How to Play

1. Press **C** to start
2. Eat food to grow and score points
3. Avoid enemies and walls
4. Use power-ups strategically
5. Survive as long as possible!

## Boss Battle

Press **B** on the ready screen to enter Boss Battle mode:
- Face a giant 3x size boss snake
- Boss aggressively chases you
- Defeat the boss to win!

## Technical

- Pure HTML5 Canvas + JavaScript
- Web Audio API for all sound effects
- No external dependencies
- ~5000 lines of game code

## Play Online

Open `index.html` in any modern web browser.
