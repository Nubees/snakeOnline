# Frontend Design Skill

When writing HTML, CSS, or JavaScript for frontend applications, you must strictly adhere to the following design constraints:

1. Zero Boilerplate: Never use default browser styling, plain white backgrounds, or unstyled system fonts.
2. Aesthetic Theme: For game development, apply a polished "retro-arcade" aesthetic. This includes dark backgrounds (e.g., #0f0f0f), glowing neon grid accents (e.g., cyan or magenta), and high-contrast primary elements.
3. Responsive Layout: The central game canvas must be perfectly centered both vertically and horizontally in the viewport using CSS Flexbox or Grid.
4. Theming Variables: Always utilize CSS variables (`:root`) for all color hex codes, font families, and spacing parameters to allow for rapid iteration.
5. Canvas Styling: Ensure the HTML5 Canvas element has a distinct, styled border (e.g., a glowing box-shadow) to visually separate the play area from the background.
